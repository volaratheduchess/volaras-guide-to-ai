# [Character Traits List](https://www.chompingatthelit.com/character-traits-list/)

**My Analysis:** It’s a very big list and can be overwhelming, however it is useful in that there are so many words. If you know what you are looking for, in that you have a synonym, you may find this helpful. Even if you don’t have an idea, you may get one by browsing here. This list of 500 words is in alphabetical order, so it allows for a mixture of traits, which is nice if you’re trying to be inspired.
