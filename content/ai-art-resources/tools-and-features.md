These are some common tools that you may need or use when making images. There is a lot of crossovers, so read through each and be aware that not all features are listed in Image Editing Mobile and PC.

# Definitions

### Image Extender/Outpainting

> Uses AI to make the scene larger. Useful for cut off body parts that didn’t fit in the frame.

### Upscaler/Enhancer

> Improves the quality of an image by recreating it with more pixels, though this makes the file size larger.

### Remove Background

> Removes the background (for various reasons) and isolates the subject. Useful for overlaying the subject on a different background.

### Compressor

> Compresses the file size to make it smaller and able to fit in size requirements.

### Retoucher

> Smooths skin, fix blemishes, adjust lighting and other touch ups.
